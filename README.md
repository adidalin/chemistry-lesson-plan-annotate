# 初中化学教案二次备课工具

帮助初中化学教师对教案进行二次备课，自动生成批注、教学反思和时间分配。

## 功能特点

| 功能 | 说明 |
|------|------|
| **智能批注** | 使用DeepSeek API分析教案内容，生成15-20条针对性教学建议 |
| **教学反思** | 自动在表格的"教学反思"单元格中填写约200字的反思内容 |
| **时间分配** | 智能分配45分钟总时长，开始和结尾短，中间长，红色标记 |

## 快速开始

### 安装依赖

```bash
pip install defusedxml requests
```

### 设置API密钥

```bash
# Windows
set DEEPSEEK_API_KEY=your-api-key

# Linux/Mac
export DEEPSEEK_API_KEY=your-api-key
```

### 使用方法

```bash
python scripts/annotate_doc.py 教案.docx
```

输出：`教案_二次备课_2026.03.25.docx`（日期为当前日期）

## 工作流程

```
1. 解包docx文档
2. 读取文档内容（段落+表格）
3. 调用DeepSeek API生成批注
4. 添加批注到文档
5. 打包输出文件
6. 生成教学反思并填写到表格
7. 智能分配时间并标记（红色）
```

## 查看结果

- **批注**：在Word中点击"审阅 → 批注"查看
- **教学反思**：直接在表格中查看（列1）
- **时间分配**：环节名称后的红色标记（如"5min"）

## 批注规范

| 项目 | 要求 |
|------|------|
| 数量 | 15-20条 |
| 字数 | 每条80字以上 |
| 分布 | 均匀分布在开头、中间、结尾 |
| 语气 | 自然流畅，像同事交流 |
| 格式 | 不用【】符号，不用"你"字 |

## 时间分配逻辑

- **有API**：调用API智能分配，开始环节短（5min），中间长（15min），结尾短（5min）
- **无API**：使用默认分配策略，确保总和为45分钟
- **已有时间**：跳过，不覆盖
- **标记格式**：红色文字，如"5min"

## 技术细节

### 表格处理
- 查找包含"教学反思"的行，在第1列填写反思内容
- 查找"环节X"开头的行，添加时间标记

### 时间分配
- 调用DeepSeek API分析环节名称和数量
- 解析API返回的时间分配（如"5,15,10,5,5,5"）
- 确保总和为45分钟
